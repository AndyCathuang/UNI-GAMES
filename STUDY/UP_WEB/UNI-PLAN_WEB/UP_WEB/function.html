UNI-PLAN的玩法说明
update:9.17

UNI-PLAN玩法说明(CN)

本游戏分有3个不同的模式。

①主线剧情模式：讲述了一个有关人类向其他星系移民的故事。

②星际战争模式：你可以在此模式中与其他玩家进行战斗，以获得胜利或失败。

③自由创造：你可以在此模式中自由创造你的世界。

UNI-PLAN Gameplay Description (EN)

The game has three different modes.

① Main story mode: tells a story about human migration to other star systems.

② Star Wars mode: you can play this mode to fight with other players to win or lose.

③ Creative mode: you can create your own world in this mode.
