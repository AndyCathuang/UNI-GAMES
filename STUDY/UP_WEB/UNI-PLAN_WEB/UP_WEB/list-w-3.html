UNI-PLAN自由创造模式玩法介绍
update:9.17

UNI-PLA自由创造模式玩法介绍(CN)



当你找到适合居住的星球后你就可以开始建造自己的家园了。

你可以在星球上建造任何你想建造的建筑，建造完成后你就可以在星球上生活了。

注：本模式提供模组玩法。

UNI-PLA Free Creative mode gameplay introduction(EN)



When you find a habitable planet, you can start building your home.

You can build whatever you want on the planet, and when you're done, you can live on the planet.

Note: This mode provides module play.
